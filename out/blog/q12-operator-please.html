The result of executing the function b is 2, and this is because of the comma is an operator in JavaScript! The comma operator is the very bottom of the operator evaluation stack, meaning a trailing comma will cause following commands to be evaluated and in this case, returned.

The comma operator is quite often seen used in for loops like so:

for(var i = 0, j = array.length; i < j; i++)

Here we're using the comma operator to chain variable assignments.

There's a lot of interesting things which can be done with the comma operator, have a read here for more ideas.
