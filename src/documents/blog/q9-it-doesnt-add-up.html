This week we've been looking at the Addition Operator and how it operates with arrays.

The result that you end up with from the log statement is:
1,2,345,6

The reason for this is because the addition operator will call the toString on the array, which will join all the items in the array into a single string, separated by a comma (although the latter is not properly defined in the spec).

 If you want to actually concatinate the arrays together you should use the concat method on the array prototype. This method can be use like this:
a.concat(b,c)

If you don't trust the provider of the arrays or you want to be sure that you're not using a concat method that has been changed you could do it like this:
Array.prototype.concat.call(a, b, c)
